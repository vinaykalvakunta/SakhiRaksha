## ğŸ Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm, yarn, or pnpm

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/sakhiraksha.git

# Navigate to the project directory
cd sakhiraksha

# Install dependencies
npm install
# or
yarn install
# or
pnpm install
```

### Development
```bash
# Start the development server
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Build
```bash
# Create a production build
npm run build
# or
yarn build
# or
pnpm build

# Start the production server
npm start
# or
yarn start
# or
pnpm start
```
